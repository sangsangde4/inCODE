# inCODE
工具链的主页网站——展示工具链主要工具、平台
技术栈架构
前端技术栈
React 18 + TypeScript + Vite
- React 18 (UI框架)
- TypeScript (类型安全)
- Vite (构建工具)
- React Router (路由管理)
- Axios (HTTP请求)
- Ant Design / Material-UI (UI组件库)
- TanStack Query (数据状态管理)
- Zustand (全局状态管理)
后端技术栈
Spring Boot + MyBatis-Plus
- Spring Boot 3.x (主框架)
- Spring Security (安全认证)
- MyBatis-Plus (ORM框架)
- Spring Validation (参数验证)
- Knife4j (API文档)
- Redis (缓存)
- JWT (Token认证)
数据库设计
MySQL 8.0 + Redis
